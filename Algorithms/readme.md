Данный курс будет посвящен изучению программирования с использованием языка **Python**. Python — это современный язык программирования, работающий на всех распространённых операционных системах.

В настоящее время существует две версии языка Python: более старая, но пока ещё более распространённая версия 2 и современная версия 3. Мы будем использовать версию 3 данного языка. Именно её необходимо установить дома, скачав данную версию с сайта [python.org](https://www.python.org).
