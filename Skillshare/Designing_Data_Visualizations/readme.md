# Designing Data Visualizations: Getting Started with Processing

## About This Class

Join Nicholas Felton – author of the Feltron Annual Reports, one of the lead designers of Facebook’s timeline, and co-founder of Daytum – to explore the world with data and design. This 90-minute class walks through the process of investigating a large amount of data, using Processing to project onto a map and polishing the visual appearance in Illustrator. It’s a great introduction to Processing and provides a data set for you to get started with right away, making this class perfect for anyone interested in design, data, geometry, or minimalism. Follow your curiosity and become fluent in presenting the data surrounding us every day.
[Course link 🔗](https://skl.sh/2Mypl0e)

## Project Description

Visualize a large set of meteorite strikes as a map

## ASSIGNMENT

Your assignment will be to transform a spreadsheet describing 34,000 meteor strikes into a visualization that places each event on the map with a marker scaled to reflect the relative sizes of the meteorites. You’ll be using both Processing and Adobe Illustrator to work through the data set that I’ve provided below for the class. You’ll find additional data in the spreadsheet that you can use to take your map further, such as color coding the years the strikes happened.

## DELIVERABLE

Upload a PNG or JPEG of your final map exported from Illustrator to the project gallery.
